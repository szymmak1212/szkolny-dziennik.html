Projekt: Szkolny Dziennik - szablony i prosta aplikacja

Zawartość archiwum:
- index.html, styles.css, app.js -> prosta aplikacja front-end (zapis w localStorage, eksport CSV)
- dziennik_template.xlsx -> szablon arkusza z przykładowymi danymi
- dziennik_template.csv -> wersja CSV szablonu

Instrukcja:
1. Rozpakuj archiwum.
2. Otwórz index.html w przeglądarce, aby korzystać z lokalnej aplikacji dziennika.
3. Możesz importować/eksportować dane jako CSV lub edytować arkusz Excel.
